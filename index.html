Umbrella Onboard: A Personalized AI Onboarding Assistant for Employees

Umbrella Onboard is an AI-powered onboarding assistant designed to help new employees navigate company policies and procedures. It uses Streamlit for the user interface, LangChain to manage prompts and context, and Retrieval-Augmented Generation (RAG) to deliver accurate, personalized responses. The assistant combines synthetic employee profiles with internal policy documents stored in a ChromaDB vector database for semantic search. Employees can ask questions about benefits, safety protocols, or leave policies and receive real-time, LLM-generated responses tailored to their role and department. Key features include contextual answers, vector-based document retrieval, streaming responses, session caching, and built-in support for common onboarding queries.

StreamPulse: High-Throughput Streaming Pipeline with Redpanda and Spark

StreamPulse is a high-throughput streaming application built with a three-node Redpanda cluster and Apache Spark for real-time data transformation. It features custom data producers in Java and Python, enabling a performance comparison. The Java producer achieved over 60,000 records/sec, while the Python version reached only 2,000 records/sec. This gap is due to Java’s compiled execution, true multi-threading, and better integration with Kafka-based systems, compared to Python’s interpreted nature and GIL-related limitations. The project highlights how language and infrastructure choices affect streaming performance.

CVSS Severity Prediction from Vulnerability Descriptions

Leveraged machine learning model to predict the CVSS (Common Vulnerability Scoring System) score of software vulnerabilities using both structured metadata and unstructured text data (e.g., CWE names and summaries). The pipeline involves comprehensive data preprocessing, text vectorization, feature engineering, and model training using a Random Forest Regressor.
